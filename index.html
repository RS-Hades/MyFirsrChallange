Tag Name What it does? It has close tag?
<head> Head It creates a space that doesn't display on the browser, it only works for details about your website that the browser needs Yes
<body> Body It creates the body of your document, all of this part will display on the browser, it's the content of your page Yes
<header> Header It creates a section into the body of your website where you can add elements, often it works as the first section of the body Yes
<main> Main It creates a section into the body of your website where you can add elements, often it works as the principal section of the body Yes
<footer> Footer It creates a section into the body of your website where you can add elements, often it works as the final section of the body Yes
<h1> H1 It creates the biggest title posible Yes
<h2> H2 It creates a title smaller than h1 but bigger than h3 Yes
<h3> H3 It creates a title smaller than h2 but bigger than h4 Yes
<h4> H4 It creates a title smaller than h3 but bigger than h5 Yes
<h5> H5 It creates a title smaller than h4 but bigger than h6 Yes
<h6> H6 It creates the smallest title posible Yes
<p> Paragraph It creates a paragraph into the document Yes
<span> Span It works as a container for text, it doesn't have an specific use due to it doesn't do anything, but you can use it as a container of text if you wanna stylize it after Yes
<div> Division It creates a section into your website where you can add elements, you can add this in any part of the body Yes
<nav> Navegation It creates a section into your website where you can add elements, usually the elements inside of this are links of navegation, between the same page or to another website Yes
<figure> Figure It creates a section into your website where you can add media content, like images, ilustrations, etc. Yes
<img> Image It creates an image into the document No
<figcaption> Figure caption It adds a caption to your media content Yes
<audio> Audio It creates an audio element, where you can reproduce an audio, into your <audio> element you have to add <source> element/s to add the audio file Yes
<source> Source It adds a media file that you've specified, it works for audio, video or pictures No
<form> Form It creates a section where you can make a form into your website Yes
<fieldset> Fieldset It creates a section into your form, you can add many fieldsets if you wanna recopilate differents kind of data Yes
<label> Label It adds text label that is used to associate it with an input element, it tells you what you have to do Yes
<input> It creates a space where the users can add data from them, it's usually used in the forms Yes
<a> Anchor It creates a link into the document, pressing it you can go to another part in the page or to another webpage Yes
<li> List It creates a list of elements Yes
<ul> Unordered list It specifies that your <li> element will be an unordered list of elements, and they will have only circles to mark each one Yes
<ol> Ordered list It specifies that your <li> element will be an ordered list of elements, and they will have numbers to mark each one Yes
<table> Table It creates a table into your document Yes
<thead> Table head It creates a section into your table where you can add table rows, usually it works as the titles or the column headers of your table Yes
<tbody> Table body It creates a section into your table where you can add table rows, usually it works as the container of the data of the table Yes
<tr> Table row It creates a row into your table where you can add a table header or a table data Yes
<th> Table header It's an data element, but it works as the main data of the table like a title, usually it would be in the <thead> element Yes
<td> Table data It creates a data element into your table Yes
<tfoot> Table foot It creates a section into your table where you can add a table row, it works as the final data of the table, like the total sum of the rest of the data Yes
<button> Button It creates a button into your website where you can add any action when you press it Yes
